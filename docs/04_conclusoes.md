# 4. Conclusões

_Navegacao: [README geral](../README.md) | [Indice de docs](README.md) | [Anterior: 3.3 Curvas de propagacao](03.3_curvas_de_propagacao.md) | [Proximo: 5. Agradecimentos](05_agradecimentos.md)_

Os resultados dos cálculos mostram que o método dos harmônicos circulares para a análise de guias de onda dielétricos retangulares fornece excelentes resultados para guias com razão de aspecto moderada. A convergência dos resultados computados foi rápida, e os resultados estão em concordância com os de Marcatili nas regiões em que suas aproximações se aplicam. Além disso, os resultados comparam-se muito bem com as curvas de Schlosser para o modo principal.

A comparação dos resultados aqui apresentados com os de Marcatili mostra que os dois métodos fornecem valores da constante de propagação normalizada, $\mathcal{P}^2$, que diferem em apenas alguns por cento para $\mathcal{P}^2 > 0{,}5$. Assim, para $\mathcal{P}^2$ nessa faixa, o método de Marcatili deve ser preferido, pois os cálculos exigidos são muito mais simples. Entretanto, para $\mathcal{P}^2 < 0{,}5$, e quando se deseja diferenciar modos em alguns dos casos quase degenerados, deve-se usar outro método.

A análise por harmônicos circulares é atraente para pequenos valores de $\mathcal{P}^2$ por causa da natureza da fronteira de ajuste. Para grande diferença de índice de refração e valores moderados de $\mathcal{P}^2$, tanto o método apresentado aqui quanto o apresentado por Schlosser podem ser utilizados.
