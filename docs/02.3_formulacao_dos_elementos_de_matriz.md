# 2.3 Formulação dos Elementos de Matriz

_Navegacao: [README geral](../README.md) | [Indice de docs](README.md) | [Anterior: 2.2 Selecao dos pontos de ajuste](02.2_selecao_dos_pontos_de_ajuste.md) | [Proximo: 2.4 Designacao dos modos](02.4_designacao_dos_modos.md)_

Os coeficientes da Equação (1) foram obtidos impondo-se continuidade aos campos elétrico e magnético tangenciais ao longo da fronteira do núcleo do guia de onda. Como cada tipo de campo possui componentes longitudinais e transversais, existem quatro tipos de equações de ajuste.

Para facilitar a análise computacional, as equações de ajuste foram escritas em forma matricial. As equações de ajuste para as componentes longitudinais dos campos são:

$E^{L,A}A = E^{L,C}C$

**Equação (6a).**

para o campo elétrico, e

$H^{L,B}B = H^{L,D}D$

**Equação (6b).**

para o campo magnético. Para os campos transversais, as equações de ajuste em forma matricial são dadas por:

$E^{T,A}A + E^{T,B}B = E^{T,C}C + E^{T,D}D$

**Equação (6c).**

para o campo elétrico, e

$H^{T,A}A + H^{T,B}B = H^{T,C}C + H^{T,D}D$

**Equação (6d).**

para o campo magnético.

As matrizes $A$, $B$, $C$ e $D$ são matrizes coluna de $N$ elementos, formadas pelos coeficientes modais $a_n$, $b_n$, $c_n$ e $d_n$, respectivamente. Os elementos das matrizes $m \times n$, isto é, $E^{L,A}$, $E^{L,C}$, $H^{L,B}$, $H^{L,D}$, $E^{T,A}$, $E^{T,B}$, $E^{T,C}$, $E^{T,D}$, $H^{T,A}$, $H^{T,B}$, $H^{T,C}$ e $H^{T,D}$, são:

$e^{L,A}_{mn} = J S$

**Equação (7a).**

$e^{L,C}_{mn} = K S$

**Equação (7b).**

$h^{L,B}_{mn} = J C$

**Equação (7c).**

$h^{L,D}_{mn} = K C$

**Equação (7d).**

$e^{T,A}_{mn} = -k_z\left(\widetilde{J}'SR + \widetilde{J}CT\right)$

**Equação (7e).**

$e^{T,B}_{mn} = k_0 Z_0\left(\widetilde{J}SR + \widetilde{J}'CT\right)$

**Equação (7f).**

$e^{T,C}_{mn} = k_z\left(\widetilde{K}'SR + \widetilde{K}CT\right)$

**Equação (7g).**

$e^{T,D}_{mn} = -k_0 Z_0\left(\widetilde{K}SR + \widetilde{K}'CT\right)$

**Equação (7h).**

$h^{T,A}_{mn} = \varepsilon_r k_0\left(\widetilde{J}CR - J'ST\right)/Z_0$

**Equação (7i).**

$h^{T,B}_{mn} = -k_z\left(\widetilde{J}'CR - \widetilde{J}ST\right)$

**Equação (7j).**

$h^{T,C}_{mn} = -k_0\left(\widetilde{K}CR - \widetilde{K}'ST\right)/Z_0$

**Equação (7k).**

$h^{T,D}_{mn} = k_z\left(\widetilde{K}'CR - \widetilde{K}ST\right)$

**Equação (7l).**

onde

$Z_0 = \left(\mu_0/\varepsilon_0\right)^{1/2}$

$\varepsilon_r = \varepsilon_1/\varepsilon_0$

$S = \sin(n\theta_m + \phi)$

$C = \cos(n\theta_m + \phi)$

com $\phi = 0$ ou $\phi = \pi/2$.

$J = J_n(h r_m), \qquad K = K_n(p r_m)$

$J' = J_n'(h r_m), \qquad K' = K_n'(p r_m)$

$\widetilde{J} = \dfrac{nJ_n(h r_m)}{h^2 r_m}, \qquad \widetilde{K} = \dfrac{nK_n(p r_m)}{p^2 r_m}$

$\widetilde{J}' = \dfrac{J_n'(h r_m)}{h}, \qquad \widetilde{K}' = \dfrac{K_n'(p r_m)}{p}$

Para $\theta_m < \theta_c$,

$R = \sin\theta_m$

$T = \cos\theta_m$

$r_m = (a/2)\cos\theta_m$

Para $\theta_m > \theta_c$,

$R = -\cos\theta_m$

$T = \sin\theta_m$

$r_m = (b/2)\sin\theta_m$

Para $\theta = \theta_c$, a fronteira no canto foi assumida perpendicular à reta radial que liga esse ponto à origem. Assim, nesse caso,

$R = \cos\left(\theta_c + \pi/4\right)$

$T = \cos\left(\theta_c - \pi/4\right)$

$r_m = \dfrac{\sqrt{a^2 + b^2}}{4}$
